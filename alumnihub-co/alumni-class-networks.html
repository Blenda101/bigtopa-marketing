What a class network runs

The reunion and
the gift behind it.

Class networks are reunion-first and giving-powered — AlumniHub handles both in one hub.

🎉
Reunion planning
Set the date, open registration, sell tickets, and run the weekend — with photos and memories saved for the next milestone year.
🎓
The class gift
Rally the class around a reunion gift to their school — college or high school — with a goal, a participation rate, and live progress everyone can see.
📇
Stay in touch
A class directory and feed that keeps everyone connected between reunions — not just in the months before one.
🙋
Class agents & volunteers
Give your class agents their own roles and peer-to-peer pages so the work — and the asks — spread across the class.
Independent, then connected

Your class hub —
linked to the whole.

A class network can run entirely on its own. When it's ready, Chapter Hub Connect links it up to the school's alumni association — collegiate or high school — so the class gift rolls into the bigger picture, without giving up control.

✓
Run solo or connected. Stand alone, or link up to the association when it suits the class.
✓
Your gift, counted twice over. Roll the class total up to the national fund while keeping your own books.
✓
Opt-in, always. The class decides what flows up — and what stays in the class.
